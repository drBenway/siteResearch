      Crawls one or more websites for url

      A pagequeue is created in the database and the url is added. Crawler then parses the first page in the queue (the url that was recieved as parameter) and adds all found links to the queue. After that the crawler looks for the next unparsed page and adds the result to the back of the queue. This repeats until the entire queue has been checked.
      The script can take some time to finish. It is advised to give php more memory and time with ini_set("memory_limit","xxxM"); set_time_limit(xxx);// time in seconds Exmpl. $url = 'https://github.com/drBenway/siteResearch/'; $scraper = new Crawler($url);
